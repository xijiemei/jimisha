# 联机测试清单

建议用两台设备，或一台电脑的两个不同浏览器测试。每轮测试都记录质量面板里的延迟、同步包大小、掉线灯状态。

## 基础连通

- 房主创建房间后，房间 ID 可复制。
- 客人输入房间 ID 后能进入大厅。
- 大厅中房主、客人、机器人在线状态显示正确。
- 在线玩家显示绿灯，掉线玩家显示红灯，机器人显示蓝灯。

## 开局流程

- 经典五猫、1v1、爆炸猫窝、至尊猫王都能进入选将。
- 所有玩家确认后能进入游戏。
- 客人只能看到自己的手牌，其他人的手牌显示为未知。
- 爆炸模式里客人只能看到牌堆数量/风险摘要，不能看到真实牌堆内容。

## 延迟观察

- 游戏开始后，客人质量面板显示“到房主 xxms”。
- 房主质量面板显示平均延迟和在线人数。
- 出牌后同步包大小有变化，并且上次同步时间会刷新。
- 同一局内延迟长期高于 260ms 时，尝试换网络或配置私有 TURN。

## 掉线与重连

- 客人断网或关闭页面后，房主端该玩家应变红灯。
- 游戏中掉线后，该玩家应暂时由机器人托管。
- 客人用相同昵称重新加入同一房间后，应回到原座位并恢复绿灯。
- 重连后继续出牌/响应不会创建重复玩家。

## 资源加载

- 首次进入菜单后，角色图和卡牌图应较快显示。
- 弱网下图片未加载完成时，应显示稳定的占位底色。
- 浏览器开发者工具 Network 中不应出现缺失图片 404。
